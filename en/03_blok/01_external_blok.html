External Blok

Add external Blok

We are going to use an existing generic business blok within our project to represent postal addresses.

Note: You can clone II_setup-project branch from AnyBlok/anyblok-book-examples repo to get ready to start with this chapter.

Add dependency

According to customers specifications we need to manage addresses, so we are going to depend on the address blok developed by the community and published in the anyblok address python package .

To get the business blok, we need to install that package by adding it in the package dependencies of the project:

diff --git a/setup.py b/setup.py

To install new package dependencies use one of the following commands:

(rooms-venv)$ make setup-dev
# or
(rooms-venv)$ python setup.py develop
# or
(rooms-venv)$ pip install -e .

To install a new blok, you may add it as a requirement to a module as shown in the next chapter or install it from command line:

(rooms-venv)$ anyblok_updatedb -c app.dev.cfg --install-bloks address

You can now add/remove addresses with the AnyBlok Python interpreter:

anyblok_interpreter -c app.dev.cfg
